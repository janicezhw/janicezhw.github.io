Introduction

GitBook使用入门

GitBook 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。

本书根据自己学习GitBook的安装及配置步骤，整理以下资源，将简单介绍如何安装、编写、生成、发布一本在线图书。
