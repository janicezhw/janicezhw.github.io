Cocoapods 私有仓库

发表于 2019-05-06 | | 阅读次数

Cocoapods仓库分为两类：公共仓库、私有仓库

公共仓库：项目中用pod管理的第三方库，我们也可以创建自己的pod仓库。

私有仓库：主要用于自己一些公共库封装管理；或者项目组件化后，各个组件独立项目，成为一个pod仓库
